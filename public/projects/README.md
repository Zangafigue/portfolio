# Captures des projets

Déposez ici les captures d'écran, **nommées par `slug`** (voir `src/data/projects.js`).
Le site les charge via `/projects/<slug>.jpg` ; en l'absence d'image, un placeholder
élégant s'affiche automatiquement (cadre navigateur + initiales + stack).

Format : **JPEG**, redimensionné à ~1280px de large (optimisé via `scripts`/Pillow).

## Projets phares (avec visuel)
- `agroconnect.jpg` — AgroConnect BF (capture web)
- `zamsma.jpg` — Zamsma (capture mobile ; affichée en « contain »)
- `gymx-v2.jpg` — GymX v2 (capture web)
- `cv-builder.jpg` — CV Builder Pro (capture web)
